loadPreset

  • loadPreset(filename: string, callback?: function): void
    • Defined in Constructor.ts:113

    Load a preset configuration consisting of model, material and sides with dimensions. Presets are stored as json-files and by default are located inside ./presets directory.

    Suppose there is a file ./presets/t-shirt.json:

    {
  "model": "t-shirt",
  "material": "cotton",
  "sides": [
    {
      "width": 250,
      "height": 400
    },
    {
      "width": 210,
      "height": 370
    }
  ]
}

    To load this preset one must call:

    printConstructor.loadPreset('t-shirt');

    Parameters

    • filename: string
    • Optional callback: function
        • (): void

        • Returns void

    Returns void
